# Publicar no GitHub — Front-end

Crie o repositório no GitHub (ex.: `Front-end-Flutter-Copa`) e substitua a URL abaixo.

## Primeira vez

```powershell
cd C:\Users\Monal\Downloads\Front-end-Flutter-Copa

git init
git branch -M main
git remote add origin https://github.com/SEU_USUARIO/Front-end-Flutter-Copa.git

git add .
git status
git commit -m "Adiciona app Flutter Copa 2026 integrado à API"
git push -u origin main
```

## Atualizar depois

```powershell
git add .
git commit -m "Sua mensagem"
git push
```
